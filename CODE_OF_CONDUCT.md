# 贡献者行为准则

## 我们的承诺

我们承诺为所有参与者提供友善、安全、无骚扰的协作环境，不因年龄、身体状况、族群、
性别认同与表达、经验水平、国籍、外貌、种族、宗教、性取向或社会经济状况而区别对待。

## 行为标准

鼓励的行为包括：尊重不同观点、给出具体且建设性的反馈、承认错误、关注社区共同利益。

不可接受的行为包括：侮辱或贬损性评论、骚扰、未经许可公开他人隐私信息，以及其他在
专业环境中可被合理视为不当的行为。

## 执行

发现不当行为时，请通过 [SECURITY.md](SECURITY.md) 中的私密联系方式报告。维护者将
公平调查，并可根据情节采取警告、临时限制或永久禁止参与等措施。所有报告者信息都会
在合理范围内保密。

本准则参考 Contributor Covenant 2.1，并以简明中文适配本项目。
