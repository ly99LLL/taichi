# yan_gua 主包

程序的全部核心代码，7 个模块。

## 文件说明（按数据流顺序）

| 阅读顺序 | 文件 | 一句话 |
|----------|------|--------|
| — | `__init__.py` | 包标记，版本号 v3.0.0 |
| 1 | `__main__.py` | **入口**：设 Java 路径 → 初始化 Taichi GPU → 预热 kernel → 打开 py5 窗口 |
| 2 | `config.py` | **所有旋钮**：粒子数、颜色、影响半径、摄像头分辨率、MediaPipe 阈值 |
| 3 | `tracking.py` | **眼睛**：打开摄像头 → CLAHE 增强 → MediaPipe 检测手和身体 |
| 4 | `motion.py` | **大脑**：从手部轨迹提取速度、曲率、纵深位移 |
| 5 | `physics.py` | **物理引擎**：Taichi GPU 并行算 6000 粒子受力 + 粒子数据管理 |
| 6 | `camera_renderer.py` | **水墨滤镜**：右下角小窗，骨架笔触 + 手指光点 + 暗角 |
| 7 | `sketch.py` | **总导演**：py5 每帧调度全部流程（读摄像头 → 分析 → 物理 → 渲染） |
